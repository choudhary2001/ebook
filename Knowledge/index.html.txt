# eBookStore Home Page - Main Landing Page

## FILE PURPOSE
The main landing page of the eBookStore website that showcases featured eBooks, provides search functionality, and serves as the entry point for users.

## STRUCTURE OVERVIEW

### 1. HTML Head Section
- **Title**: "eBookStore - Premium Digital Books Collection"
- **Meta Description**: SEO-optimized description for search engines
- **Viewport**: Mobile-first responsive meta tag
- **Stylesheets**: TailwindCSS CDN and custom styles
- **Fonts**: Google Fonts (Inter and Merriweather)

### 2. Header Component
- **Logo**: eBookStore branding with SVG icon
- **Navigation**: Desktop navigation menu (Home, Categories, About, Contact)
- **Theme Toggle**: Light/dark mode switcher
- **Mobile Menu**: Hamburger menu for mobile devices

### 3. Hero Section
- **Gradient Background**: Primary color gradient
- **Main Heading**: "Discover Your Next Favorite eBook"
- **Subheading**: Descriptive text about the collection
- **CTA Button**: "Explore eBooks" with scroll-to functionality
- **Statistics**: Display of available eBooks count

### 4. Search Section
- **Search Input**: Real-time search functionality
- **Category Filters**: Quick filter buttons for different categories
- **Responsive Design**: Adapts to all screen sizes

### 5. Featured Books Section
- **Loading State**: Skeleton placeholders while data loads
- **Books Grid**: Responsive grid layout (1-4 columns based on screen size)
- **Book Cards**: Individual book display components
- **No Results**: Empty state when no books match search

### 6. Newsletter Section
- **Email Subscription**: Newsletter signup form
- **Background**: Primary color accent background
- **Privacy Note**: User privacy assurance text

### 7. Footer
- **Company Info**: Logo, description, social links
- **Quick Links**: Navigation shortcuts
- **Support Links**: Help and policy links
- **Copyright**: Legal information

### 8. Back to Top Button
- **Fixed Position**: Bottom-right corner
- **Smooth Scroll**: Animated scroll to top
- **Visibility Toggle**: Shows/hides based on scroll position

## KEY FEATURES

### Search and Filtering
- **Real-time Search**: Instant results as user types
- **Multi-field Search**: Searches title, author, category, description
- **Category Filters**: Quick filter buttons for each category
- **Debounced Input**: 300ms delay to optimize performance
- **URL Parameters**: Supports ?search= parameter for direct links

### Responsive Design
- **Mobile-first**: Designed for mobile devices first
- **Breakpoints**: 
  - xs: 475px
  - sm: 640px
  - md: 768px
  - lg: 1024px
  - xl: 1280px
- **Grid System**: Responsive grid that adapts from 1 to 4 columns
- **Touch-friendly**: Large tap targets for mobile devices

### Performance Optimizations
- **Lazy Loading**: Images load only when in viewport
- **Skeleton Loading**: Placeholder content while data loads
- **Debounced Search**: Prevents excessive filtering operations
- **Intersection Observer**: Efficient scroll-based animations
- **Image Placeholders**: SVG placeholders prevent layout shift

### Accessibility Features
- **Semantic HTML**: Proper heading hierarchy and landmarks
- **ARIA Labels**: Screen reader support for interactive elements
- **Keyboard Navigation**: Full keyboard accessibility
- **Color Contrast**: WCAG compliant color ratios
- **Focus Indicators**: Visible focus states for all interactive elements

## DATA INTEGRATION

### eBooks JSON Structure
```javascript
// Expected data format from data/ebooks.json
{
  id: number,
  title: string,
  author: string,
  price: string,
  category: string,
  description: string,
  cover: string,
  rating: number
}
```

### Data Loading Process
1. **Fetch**: Loads data/ebooks.json via fetch API
2. **Parse**: Converts JSON to JavaScript array
3. **Filter**: Creates filteredBooks array for search/filter
4. **Render**: Generates HTML for each book
5. **Error Handling**: Shows error message if loading fails
